### 01-5. 헤이 지니! 소리 질러!

```
>> print("Hello World")
Hello World

>> print(5)
5

>> print("5")
5

>> print(3 + 5)
8

>> print("3 + 5")
3 + 5

>> print("3 + 5 =", 8)
3 + 5 = 8
```
